Marketing Campaign Performance Dashboard
Overview

This project analyzes a bank marketing campaign using Power BI, focusing on identifying key drivers of customer conversion.

Key Questions
Which customer segments convert best?
Does financial status impact conversion?
How does campaign frequency affect performance?
Key Insights
Conversion rates increase with age, with 60+ customers showing the highest performance.
Customers with higher balances demonstrate higher conversion rates.
Conversion declines after multiple contact attempts, indicating diminishing returns.
