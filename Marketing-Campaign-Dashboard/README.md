# Marketing Campaign Effectiveness Analysis in Power BI

## Project Overview
This project analyzes a **bank marketing campaign** using Power BI. The dataset includes various customer and campaign attributes such as age, financial balance, and campaign contact method. The objective is to segment customers, understand conversion patterns, and identify ways to optimize the marketing campaign.

## Key Insights
1. **Age Group Insights**:  
   - Conversion rates increase with age, with customers aged **60+** showing the highest conversion rates.

2. **Balance Insights**:  
   - Customers with **higher financial balances** are more likely to convert, indicating the importance of targeting wealthier segments.

3. **Campaign Insights**:  
   - **Diminishing returns** occur after multiple contact attempts. Conversion rates drop as the number of contact attempts increases, suggesting the need to limit excessive outreach.

## How to Run the Project
1. Download the Power BI file (**PowerBI_Dashboard.pbix**).
2. Open the file in **Power BI Desktop**.
3. Explore the dashboard for insights and recommendations.

## Technology Stack
- Power BI Desktop
