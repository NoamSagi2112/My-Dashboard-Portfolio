# Global Electronics Retail Dashboard (Power BI)

## Overview
Interactive dashboard analyzing revenue, orders, products, and categories for a global electronics retailer.

## Features
- KPIs: Revenue, Orders, Quantity, AOV with YoY comparison
- Revenue Trend by Quarter-Year
- Top Product Categories and dynamic Top 5 products
- Chart-to-chart interactivity (click Category/Month → Top Products updates)

## Tools
- Power BI Desktop
- DAX measures

## Insights
- Revenue concentrated in top categories
- Top products drive majority of revenue
- Dynamic interactivity allows exploration by category and quarter
